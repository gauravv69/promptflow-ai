# PromptFlow AI

A modern AI SaaS dashboard built with React, TypeScript, and Tailwind CSS.

## Features
- Chat interface (ChatGPT-like)
- Prompt history
- Saved responses
- Dark mode UI
- Scalable architecture

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS
- Zustand + React Query
- Framer Motion

## Setup
npm install
npm run dev
